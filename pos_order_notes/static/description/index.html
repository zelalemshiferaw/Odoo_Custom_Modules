Odoo POS Order Notes

Now add notes to individual orderlines and complete order in POS!

Odoo POS Order Notes module allows you to add internal notes for administrator's reference. It facilitates the user to add order notes to individual order lines as well as on complete order. These custom messages will be displayed with the order in POS.

How does it work?
  • Sellers can add notes for each orderline in POS.
  • They can add a note for the complete order.
  • Sellers can activate or deactivate the order note feature.
  • They can even print the order note on the receipt if he wants.
